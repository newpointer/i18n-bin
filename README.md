# nullpointer-i18n-bin

> Скрипты интернационализации

##Окружение

Установить:

* node.js 0.10.x+
* npm 1.3.x+
* grunt-cli `npm install grunt-cli -g`

##Инициализация

Установка зависимостей

    npm install


##Сборка

Проверка кода, тесты

    grunt build


##Запуск тестов

Только тесты

    grunt test


##Дистрибутив

Установка проверка кода, тесты, дистрибутив, документация

    grunt dist


##Очистка

Удаление зависимостей

    grunt clean
